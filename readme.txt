How to compile:
1. Open "EGP-410/GameAI/component steering SDL
2. Find and open steering.sln
3. Make sure "steering" is the Startup Project
4. Press F5/Ctrl+F5 to run the program

Keyboard Inputs:
a: Adds 10 Boids
d: Deletes random Boid
s: Changes Edit mode to Separation
c: Changes Edit mode to Cohesion
g: Changes Edit mode to Group Align
up/down arrow: Changes Weight of current Edit mode
left/right arrow: Changes radius of current Edit mode
Enter: Saves Flocking Data
Esc: Exits Simulation
